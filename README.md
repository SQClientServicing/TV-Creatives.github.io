# Food Creative Builder

## Overview
Host-ready HTML creative builder for SmartQ announcement creatives.

## Features
- 1920x1080 canvas
- Background template selection
- Logo upload
- Editable body copy
- Poppins typography
- PNG/JPG export
- SmartQ favicon

## Local Usage
Open `index.html` in a modern browser.

## Deployment
### GitHub Pages
1. Create a repository.
2. Upload all files.
3. Enable Pages from the main branch.

### Netlify
Drag and drop this folder or ZIP.

### Vercel
Import the folder as a static site.
